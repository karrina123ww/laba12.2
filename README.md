Описание тест-кейсов для лабораторной работы
Автоматизированные тесты для сайта https://the-internet.herokuapp.com

Описание тест-кейсов:
Тест 1. Отметка чекбокса
Проверка возможности отметить чекбокс на странице Checkboxes. Ожидается, что первый чекбокс успешно изменяет своё состояние с неотмеченного на отмеченное, и тест подтверждает, что чекбокс выбран.

Тест 2. Выбор опции из выпадающего списка
Проверка возможности выбора опции из выпадающего списка Dropdown. Ожидается, что при выборе опции "Option 2" она становится активной, и текст выбранной опции соответствует ожидаемому значению "Option 2".

Тест 3. Заполнение и отправка формы авторизации
Проверка успешной авторизации с валидными учётными данными (tomsmith / SuperSecretPassword!). Ожидается, что после входа появляется сообщение «You logged into a secure area», а после нажатия кнопки Logout появляется сообщение «You logged out of the secure area», подтверждающее успешный выход из системы.
